Изменение внешнего вида символов и абзацев называется форматированием.

Для запуска программы-кодировщика введите в поле слово Кодировка.

ABC - этими буквами в Америке обзначают алфавит.

<STRONG></STRONG> - используется для выделелния важных фрагментов текста полужирным шрифтом.

<VAR></VAR> используется для отметки имен переменных.
Обычно такой текст выделяется курсивом. Например:
Пусть задана функция: y=2x

<ADDRESS></ADDRESS> - используется при выделелении почтового адреса

курсивом.

<STRIKE></STRIKE> - используется для зачеркивания текста.
